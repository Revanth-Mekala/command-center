    Integrations
    Required to connect Google Calendar.
    How to get this
    1. Go to console.cloud.google.com and create or select a project.
    2. In the left menu go to APIs & Services → Library, search for Google Calendar API, and enable it.
    3. Go to APIs & Services → Credentials and click + Create Credentials → OAuth client ID.
    4. Set application type to Web application.
    5. Under Authorised JavaScript origins add http://localhost (and any other origin you use, e.g. http://localhost:3131).
    6. Click Create — copy the Client ID (ends in .apps.googleusercontent.com) and paste it below.
